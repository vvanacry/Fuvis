# Fuvis
AI-критик идей — это инструмент для честной и трезвой оценки проектов. Он разбирает идеи без мотивационного шума, показывает слабые места, риски и иллюзии, а если идея жизнеспособна — помогает улучшить её и подготовить к обсуждению.

-Для пользования написать e-mail на сайте, скину пробную версию. (https://fuvis.netlify.app/)
